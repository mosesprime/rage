# Rage Attributes

## About
A better pre-processor hint.

## Standard Attributes

Attributes that come with default with the language.

- `#[inline]`, `#[inline(false)]`, or `#[inline(true)]` : Suggest, restrict, or force the compiler to inline the following code. `inline` expands to this.
- `#[public]` : Exported and/or accessable. `pub` expands to this.
- `#[mutable]` : Can be modified. `mut` expands to this.
